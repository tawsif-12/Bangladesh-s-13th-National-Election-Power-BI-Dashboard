# 📊 Bangladesh’s 13th National Election – Power BI Dashboard (Practice Project)

## 📌 Project Overview

This project is an interactive Power BI dashboard developed as part of my hands-on learning in **Data Analytics, Data Modeling, and Data Visualization**.

The objective of this report is to present election results in a clear, insightful, and user-friendly way so that users can quickly understand overall performance, compare district-wise outcomes, and analyze party-wise seat distribution.

This is a **practice project built using dummy data** and does not represent real election results.

---

## 🎯 Project Goals

* Practice building a complete Power BI report from scratch
* Create meaningful KPIs using DAX
* Improve dashboard design and layout skills
* Apply visual storytelling techniques
* Develop a portfolio-ready project

---

## 📊 Dashboard Features

### 🔹 KPI Summary

* Total Seats
* Total Parties
* Total Candidates
* Total Centers

These KPIs provide a high-level snapshot of the election scenario.

### 🔹 Party-wise Seat Distribution

A pie chart that shows how many seats were won by each party, helping users quickly identify the dominant political group.

### 🔹 District-wise Party Performance

A stacked column chart to compare which party performed best in each district.

### 🔹 Referendum Analysis

A donut chart representing the proportion of **Yes vs No** votes for quick public opinion analysis.

### 🔹 Detailed Result Table

Includes:

* Seat number & constituency
* Winning party
* Top candidate
* Total votes received

This allows deeper, record-level exploration.

## 🛠 Tools & Technologies

* Microsoft Power BI
* DAX
* Power Query
